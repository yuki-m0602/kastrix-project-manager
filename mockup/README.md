# mockup/

UI 実装前のプロトタイプ・デザイン確認用モックアップを格納するディレクトリです。

## 方針

| 項目 | 内容 |
|------|------|
| 用途 | プロトタイプ・デザイン確認に限定 |
| 本番 | `ui/` が本番 UI。mockup は参照用 |
| 統合 | ui に統合しない。参照用として別管理 |

## 使い方

- 新機能の UI を実装する前に、`agent_docs/ui.md` に従いモックアップを確認する
- モックアップは CDN 等でスタンドアロン表示可能（Tauri 不要）
- `bak/` は過去バージョンのバックアップ

## 主なファイル

- `index.html` — メイン画面のプロトタイプ
- `team-feature.html` — チーム機能の UI モックアップ
